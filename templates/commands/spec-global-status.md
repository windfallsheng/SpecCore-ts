# /spec-global-status - 查看全量层状态

## 命令说明

基于 `GLOBAL/INDEX.md` 和 `GLOBAL/PROJECTS/`，展示全量层状态。

**数据来源**：
- 项目总览 / 需求总览 / 期次关联 → 来自 `GLOBAL/INDEX.md`
- 项目详情 / 需求详情 → 穿透到 `GLOBAL/PROJECTS/{project}/REQUIREMENT.md`

## 使用方式

### 标准命令格式
```bash
/spec-global-status
/spec-global-status --project=用户服务
```

### 自然语言格式
```bash
全量状态
查看全量层
用户服务的全量状态
```
