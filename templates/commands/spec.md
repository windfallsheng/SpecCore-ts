# /spec - 智能入口（意图识别引擎）

## 命令说明

用户输入自然语言描述，AI 自动识别意图，匹配最合适的 SpecCore 命令并执行。

**核心理念**：把 28 个命令的复杂度隐藏在 AI 背后，用户只需要用自然语言表达意图。

**适用场景**：
- 不知道用哪个命令
- 想要快速操作，不想记命令名
- 用自然语言更自然地表达需求

## 使用方式

```text
/spec 做一个用户登录功能，支持手机号+密码
/spec 修复登录超时问题
/spec 查看进度
/spec 开始干活
/spec 把登录改成验证码登录
/spec 审查一下
/spec 参考一下之前的认证实现
```

## AI 执行流程

### 第一步：加载意图识别引擎

加载 `templates/skills/intent-recognition/SKILL.md`，启用意图识别能力。

### 第二步：意图识别

按以下流程处理用户输入：

```text
用户输入
    │
    ▼
┌─────────────────────────────────────────────────────────────┐
│  意图识别引擎                                               │
│  ┌───────────────────────────────────────────────────────┐  │
│  │ 1. 加载上下文（current_iteration/current_task/...）   │  │
│  │ 2. 关键词匹配（12 种意图类型，100+ 关键词）           │  │
│  │ 3. 上下文感知增强（当前期次/任务/状态）               │  │
│  │ 4. 置信度计算（优先级 + 命中 + 模式 + 上下文）       │  │
│  │ 5. 参数提取（名称、类型、描述等）                     │  │

└─────────────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────────────┐

    │
    ▼
┌─────────────────────────────────────────────────────────────┐

│  ┌───────────────────────────────────────────────────────┐  │
│  │ 1. 展示操作预览 → 2. 确认后执行 → 3. 实时反馈 → 4. 结果 │  │

└─────────────────────────────────────────────────────────────┘
```

### 置信度分级

| 置信度 | 行为 |
| :---: | :--- |
| **≥ 80%** | 直接展示操作预览，确认后执行 |
| **50-80%** | 展示 2-3 个候选命令，让用户选择 |
| **< 50%** | 引导用户补充说明，给出示例 |

### 上下文感知增强

| 用户输入 | 有上下文时 | 无上下文时 |
| :--- | :--- | :--- |
| "开始干活" | → 执行当前期次待开发任务 | → 提示先创建期次 |
| "进度怎么样了" | → 显示当前期次进度 | → 提示当前无活跃期次 |
| "审查一下" | → 审查当前 Task | → 提示指定要审查的 Task |
| "继续开发" | → 断点续传 | → 提示无中断记录 |
| "参考一下" | → 搜索与当前 Task 相关的模式 | → 提示指定参考目标 |

```text
┌─────────────────────────────────────────────────────────────┐
│              用户自然语言输入层                              │
│   "进度"  "开始干活"  "做一个登录"  "审查一下"  "改成..."  │
└─────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────┐
│           意图识别引擎（intent-recognition SKILL）          │
│   关键词匹配 → 上下文感知 → 置信度计算 → 参数提取           │
└─────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────┐
│              命令匹配与执行                                  │
│   高置信度 → 直接执行  中置信度 → 展示候选  低置信度 → 澄清 │
└─────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────┐
│          现有命令体系（28 个命令，全部保留）                │
│   spec-progress  spec-execute  spec-goal  spec-review ...  │
└─────────────────────────────────────────────────────────────┘
```

> 💡 **核心变化**：用户不需要知道 28 个命令的存在，只需用自然语言表达意图。AI 负责翻译并执行。
> 现有命令和别名系统全部保留，熟练用户可继续直接使用。
