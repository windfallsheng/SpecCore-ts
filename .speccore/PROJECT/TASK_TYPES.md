# 任务类型定义

> SpecCore 支持多种任务类型，不同类型的任务有不同的 Spec 章节要求。

## 类型列表

| 类型 | 标识 | 说明 | 适用场景 |
| :--- | :--- | :--- | :--- |
| **功能开发** | `feature` | 新增业务功能 | 新功能开发、模块新增 |
| **Bug 修复** | `bugfix` | 修复线上问题 | 缺陷修复、紧急补丁 |
| **技术调研** | `research` | 技术选型、方案评估 | 技术预研、方案对比 |
| **性能优化** | `optimization` | 性能提升、代码重构 | 接口优化、架构调整 |
| **数据迁移** | `migration` | 数据迁移、数据库变更 | 数据库升级、数据同步 |
| **文档编写** | `document` | 技术文档、API 文档 | 文档补充、知识沉淀 |

## 各类型的必需章节

### `feature` 功能开发

| 文件 | 必需章节 |
| :--- | :--- |
| REQ.md | 需求描述、业务规则、验收标准 |
| TECH.md | 方案概述、接口设计、核心类设计 |
| TASK.md | 变更履历、上下文清单、产出物清单、AC |

### `bugfix` Bug 修复

| 文件 | 必需章节 |
| :--- | :--- |
| REQ.md | 问题描述、复现步骤、期望结果 |
| TECH.md | 根因分析、修复方案、回归范围 |
| TASK.md | 变更履历、上下文清单、修复产出、验证方法 |

### `research` 技术调研

| 文件 | 必需章节 |
| :--- | :--- |
| REQ.md | 调研目标、要回答的问题 |
| TECH.md | 调研方法、候选方案对比、评估维度 |
| TASK.md | 变更履历、上下文清单、调研报告、建议方案 |

### `optimization` 性能优化

| 文件 | 必需章节 |
| :--- | :--- |
| REQ.md | 当前问题、优化目标、期望指标 |
| TECH.md | 优化方案、实施步骤、回滚方案 |
| TASK.md | 变更履历、上下文清单、优化产出、性能对比 |

### `migration` 数据迁移

| 文件 | 必需章节 |
| :--- | :--- |
| REQ.md | 迁移背景、迁移范围、兼容性要求 |
| TECH.md | 迁移步骤、验证方案、回滚计划 |
| TASK.md | 变更履历、上下文清单、迁移脚本、验证报告 |

### `document` 文档编写

| 文件 | 必需章节 |
| :--- | :--- |
| REQ.md | 文档目标、受众、范围 |
| TECH.md | 文档结构、引用资料 |
| TASK.md | 变更履历、上下文清单、文档产出、审阅人 |
