# Task-001 任务CRUD - web 前端需求

> 目标平台：web | 关联后端：Task-001

## 页面列表

| 页面 | 路由 | 说明 |
| :--- | :--- | :--- |
| 任务列表 | /tasks | 分页表格 + 新建按钮 |
| 任务详情 | /tasks/:id | 查看/编辑表单 |
| 新建任务 | /tasks/new | 创建表单 |

## 组件树

- TaskTable: 分页表格 + 操作列(编辑/删除)
- TaskForm: 新增/编辑共用表单
- TaskDeleteConfirm: 删除确认弹窗

## 验收标准

- [ ] 列表默认 20 条/页，支持翻页
- [ ] 新增/编辑用同一表单，校验通过后提交
- [ ] 删除弹出确认框，成功后刷新列表
- [ ] 操作成功显示 toast，失败显示错误
- [ ] 表格 loading 状态正确显示
