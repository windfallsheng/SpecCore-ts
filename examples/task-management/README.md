# SpecCore 示例 — 任务管理系统

> **完整演示：需求 → Spec → 生成代码的全流程。**

---

## 示例概述

用 SpecCore 从零开发一个「任务管理系统」，包含：

| REQ | 需求 | 生成产物 |
| :--- | :--- | :--- |
| REQ-001 | 任务 CRUD | Controller.java + Service.java + Repository.java + .vue |
| REQ-002 | 任务分配 | 同上 |
| REQ-003 | 任务筛选 | 同上 |

产物包含：
- `期次-Q3/` — Spec 文件（REQ.md / TECH.md / TASK.md / API_CONTRACT.yaml）
- `期次-Q3/Task-*/backend/` — 生成的 Java 代码骨架
- `期次-Q3/Task-*/frontend/web/` — 生成的 Vue 组件骨架
- `.speccore/` — 全局配置、索引、上下文

---

## 快速复现（从头跑一遍）

```bash
# 1. 进入示例目录
cd examples/task-management

# 2. 重置（如需重新生成）
rm -rf .speccore 期次-Q3 && mkdir -p src/main/java/com/example

# 3. 初始化
speccore init

# 4. 导入代码
speccore import --project=task-api --path=./src --type=backend

# 5. 创建期次
speccore iteration create --name=Q3

# 6. 创建 Task
speccore new-task --name="任务CRUD" --platforms=web --type=feature
speccore new-task --name="任务分配" --platforms=web --type=feature
speccore new-task --name="任务筛选" --platforms=web --type=feature

# 7. 执行（生成代码骨架）
speccore execute --all --force

# 8. 查看产物
ls 期次-Q3/Task-001-任务CRUD/backend/
```
