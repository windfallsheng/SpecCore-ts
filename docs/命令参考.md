## 🌐 初始化与导入

| 命令 | 别名 | 功能 | 参数 |
| :--- | :--- | :--- | :--- |
| `speccore init` | `in` | 初始化 Speccore 项目 | `--force` |
| `speccore word2spec` | `w2s` | Word(.docx/.doc) → SpecCore Markdown | `--file <path>` `--iteration <name>` `--platform <name>` |
| `speccore import` | `imp` | 导入项目到全量层 | `--project <name>` `--path <path>` `--type <type>` `--scope <scope>` `--ignore <pkgs>` `--update` |

```bash
speccore init
speccore init --force
speccore word2spec -f api-prd.docx -i Q3 -p 后台
speccore word2spec -f web需求.doc -i Q3 -p Web
speccore import --project=user-service --path=../backend --type=backend
speccore import --project=web --path=../frontend --type=web --update
```

## ⚡ 执行与调度

| 命令 | 别名 | 功能 | 参数 |
| :--- | :--- | :--- | :--- |
| `speccore plan` | `pl` | 智能调度方案（DAG） | `--iteration <name>` |
| `speccore execute` | `ex` | 执行开发任务 | `--all` `--task <id>` `--batch-size=<n>` `--resume` `--dry-run` `--force` `--interactive` `--platform <name>` `--iteration <name>` |

```bash
speccore plan --iteration=期次-2026-07
speccore execute --task=Task-001 --force
speccore execute --all --batch-size=3 --force      # 分批执行
speccore execute --all --dry-run                    # 预览
speccore execute --resume                           # 断点续传
speccore execute --all --interactive                # 交互选择
```

| :--- | :--- | :--- | :--- |
| `speccore update` | `up` | 更新 Task 属性 | `--task=<id>` `--status=<s>` `--priority=<p>` `--assignee=<name>` |
| `speccore backup` | `bk` | 备份当前状态 | `--list` `--restore=<name>` |
| `speccore completion` | `cm` | 生成 Shell 补全 | `[bash\|zsh]` |
| `speccore hooks` | `hk` | 安装 Git hooks | — |
| `speccore current` | `cr` | 查看分支关联任务 | `--commit` `--pr` |
| `speccore diff` | `df` | 对比期次/基线差异 | `--source=<name>` `--target=<name>` |
| `speccore trace` | `tr` | REQ→Task→Code 追溯链 | `--req=<id>` `--task=<id>` `--full` |

```bash
# update
speccore update --task=Task-001 --status=completed
speccore update --task=Task-001 --priority=high --assignee=张三

# backup
speccore backup
speccore backup --list
speccore backup --restore=2026-07-11T10:00:00

# completion
speccore completion bash > /usr/local/etc/bash_completion.d/speccore

# hooks
speccore hooks install

# current
speccore current
speccore current --commit
speccore current --pr

# diff
speccore diff --source=期次-2026-07 --target=期次-2026-08

# trace
speccore trace --req=REQ-001
speccore trace --task=Task-001
speccore trace --full
```
