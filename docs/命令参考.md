## 🆕 v4.7~v5.3 新增命令（7 个）

| 命令 | 别名 | 说明 | 参数 |
| :--- | :--- | :--- | :--- |
| `speccore update` | `up` | 更新 Task 属性（状态/优先级/负责人）| `--task=<id>` `--status=<s>` `--priority=<p>` `--assignee=<name>` |
| `speccore backup` | `bk` | 备份当前状态 | `--list` `--restore=<name>` |
| `speccore completion` | `cm` | 生成 Shell 补全脚本 | `[bash\|zsh]` |
| `speccore hooks` | `hk` | 安装 Git hooks (pre-commit + pre-push) | — |
| `speccore current` | `cr` | 查看当前分支关联任务 | `--commit` `--pr` |
| `speccore diff` | `df` | 对比期次/基线差异 (v5.3) | `--source=<name>` `--target=<name>` |
| `speccore trace` | `tr` | REQ→Task→Code 追溯链 (v5.3) | `--req=<id>` `--task=<id>` `--full` |

### update 示例
```bash
speccore update --task=Task-001 --status=completed
speccore update --task=Task-001 --priority=high --assignee=张三
```

### backup 示例
```bash
speccore backup                         # 创建备份
speccore backup --list                  # 查看备份列表
speccore backup --restore=<timestamp>   # 恢复备份
```

### completion 示例
```bash
speccore completion bash > /usr/local/etc/bash_completion.d/speccore
source ~/.bashrc
```

### hooks 示例
```bash
speccore hooks install     # 安装 pre-commit + pre-push hooks
```

### current 示例
```bash
speccore current           # 查看当前分支关联任务
speccore current --commit  # 生成 commit message
speccore current --pr      # 生成 PR description
```
