# SpecCore 速查卡

> 常用命令 + 安全口诀，一页全掌握。

---

## 🔧 终端命令（speccore 开头）

```bash
speccore init                         # 初始化项目
speccore import --project=<n> --path=<p>  # 导入项目到全量层
speccore iteration create --name=<n>  # 创建期次
speccore new-task --platforms=web,h5  # 创建多平台 Task
speccore execute --all --batch-size=3 # 分批执行
speccore validate                     # 合规检查
speccore progress --platform=web      # 查看进度
speccore update --task=T-001 --status=completed  # 更新状态
speccore diff --source=A --target=B   # 对比差异
speccore trace --req=REQ-001          # 追溯链路
speccore backup                       # 备份当前状态
```

## 📁 文件操作安全口诀

```
✅ 安全（直接改）          ❌ 危险（用命令）
REQ.md  需求描述          PROJECT_GRAPH.md  表格
TECH.md 技术方案          .task-type        类型
*.md    注释/TODO         API_CONTRACT.yaml 接口
                          .json 数据文件
⚠️ 慎改（改完立即同步）
GLOBAL/REQUIREMENT.md → speccore sync-global
```
