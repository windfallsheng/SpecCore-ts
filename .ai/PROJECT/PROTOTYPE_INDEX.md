# 原型索引

> SpecCore 框架的设计稿和原型文件位置。

## 1. 设计稿位置

目前 SpecCore 框架为纯文本框架，无 UI 设计稿。所有视觉规范通过 Markdown 文档描述。

## 2. 与 Feature 的关联

业务项目的原型文件应存放在：
- `Feature-XXX/backend/prototypes/` — 后端设计稿/数据模型图
- `Feature-XXX/frontend/prototypes/` — 前端 UI 设计稿

## 3. 设计规范

- 原型格式：PNG / JPG / Figma 链接
- 命名规范：`{页面名}-{分辨率}.{ext}`（如 `login-1920x1080.png`）
