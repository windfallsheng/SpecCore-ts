# 项目全景

## 1. 项目背景

SpecCore 是一个 Spec-Driven Development（规范驱动开发）工程框架，专为 AI 原生团队设计。解决 AI 辅助开发中的上下文丢失、知识不沉淀、前后端协作困难等核心痛点。

## 2. 项目目标

提供一套可落地的规范驱动研发方法论，包含文件模板、命令体系、裁决规则和知识沉淀机制，让 AI 在正确的时刻拿到正确的信息。

## 3. 项目范围

### 3.1 包含范围
- 方法论框架设计
- 文件模板体系（全局层 + 期次层 + Feature 层）
- 15 个 Slash Command 命令体系
- Skill 封装（前后端开发流程）
- 初始化脚本和同步脚本
- 多工具适配（WorkBuddy / Qcoder / Trae）

### 3.2 不包含范围
- 不实现为 CLI 工具（保持纯 Markdown 框架）
- 不绑定特定 IDE
- 不包含业务代码实现
