## 模式文件模板

新建模式文件时，按以下结构编写（或直接复制 [_TEMPLATE.md](./_TEMPLATE.md)）：

```markdown
# P-XXX: {模式名称}
## 1. 适用场景
## 2. 技术栈锁定
## 3. 核心实现片段
## 4. 踩坑记录
## 5. 复用 Checklist
```
